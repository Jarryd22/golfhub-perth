# GolfHub Perth

A native Windows desktop app for finding public golf bookings across Perth and Peel, with weather, official booking links, live availability feeds, and a shared four-week cache.

## Status

GolfHub Perth v4 is built with Python and PySide6/Qt. GitHub Actions refreshes the public booking cache every 10 minutes across a rolling 28-day window.

See `README_PRO_APP.txt` and `RESEARCH_AND_TEST_REPORT.md` for setup, usage and verification details.
